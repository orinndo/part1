使い方（GitHub Pages）
1) このフォルダの中身を GitHub リポジトリ直下にアップロード
2) Settings → Pages → main / (root) を選択
3) 表示されたURLをQRコード化（QRは1枚）

音声追加方法
- audio フォルダに、次の名前で音声(m4a)を追加してください
  audio01.m4a ～ audio17.m4a
- 例: 「2番の音声」→ audio/audio02.m4a
- 追加後に commit すれば、同じQRのまま音声が増えます

注意
- iPhoneは自動再生が制限されるため、ボタンで1回タップして再生します。
